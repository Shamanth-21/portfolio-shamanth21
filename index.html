Benchmarking SQL‑Generating LLMs (NL→SQL)

Built a rigorous, reproducible evaluation harness for text‑to‑SQL using exact‑match and execution accuracy. Compared Mistral‑7B‑Instruct and CodeLLaMA‑7B across plain, schema‑aware, and RAG prompts; added LoRA fine‑tuning for CodeLLaMA on Chinook to test adaptation.

Result: RAG markedly cut hallucinations; CodeLLaMA produced more structurally correct SQL, while Mistral aligned better with natural language. Outcome: a practical vetting workflow teams can run offline.
  • Dual metrics prevent over‑optimistic scoring and surface logically wrong-but-valid queries.
  • Prompt builder retrieves schema/examples per query to ground generations.
  • Artifacts and runs logged for apples‑to‑apples model comparisons.

Demand Signal Classification with LoRA‑tuned Phi‑1.5

Converted noisy B2B sales notes into actionable demand categories using parameter‑efficient fine‑tuning (LoRA) on Phi‑1.5 with 4‑bit quantization—engineered to run on a Colab T4.

Result: Validation accuracy 100% on held‑out synthetic notes; precision/recall/F1 all 1.00. Next: expand with real labeled notes + HITL validation.
  • Synthesized ~2k notes with realistic noise (typos, spacing, urgency markers) to mirror production.
  • ~1% trainable parameters via LoRA; early stopping to manage overfitting risk.
  • Designed review loop for high‑stakes decisions before deployment.
